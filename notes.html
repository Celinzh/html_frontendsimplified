Inline vs Block

~ Blockelements start on a new line and take the full width available

~ Inline elements do not start on a new line and take only the necessary width

~ So me elements are default inline elements and some elements are default block elements Eg. Button is a inline element. It decides whether the elements would take up a whole line or it will just use as much space as they need

Some examples:

- Block Level: #div#, #h1# till #h6#, #p#

- Inline Level: #span#, #a#, #button#
